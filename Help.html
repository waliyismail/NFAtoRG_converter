RG to NFA Help

  • First, enter all the Regular Grammars.
  • Note to input at least 3 alphabet and ε
  • Enter -> for arrow, | as or, and "e" for ε.
  • Reminder, first state is consider starting state, last state is consider final accepting state and can only have one final state.
  • Example :

    • A->aR

    R->bR|aS

    S->bT

    T->R|e

  • Click on "Convert to NFA" button to display the Formal Description and create table based on your input.
  • Click on "To NFA without ε" button to display the Transition Table without ε .
  • Then, enter strings of input to check.
  • Note that it can check minimum 5 strings at once.
  • Example :

    • e

    aaa

    bbbb

    abbbbab

    aab

NFA to RG Help

  • enter "epsilon" for ε.
  • first, enter all the formal description.
  • click on "transition table" button to create table based on your input.
  • a table will show and you need to fill up the transition table for the states.
  • if you want to change the formal description, you can fill the new one and click "refresh table" to refresh the table.
  • next, fill in the check string and click "check" to check your input.
  • if you are statisfied, click on "convert to regular grammar" to convert your nfa to rg.
  • please don't put weird input.
  • please fill up all input before you convert.
  • please don't put extra spaces if you don't need
